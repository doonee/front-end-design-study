display: inline - 기본값으로, 요소를 inline 요소처럼 표시합니다.
따라서 앞뒤로 줄바꿈 되지 않습니다.

display: block - block 요소를 block 요소처럼 표시합니다.
따라서 요소 앞 뒤로 줄바꿈이 되고 수직으로 정렬 됩니다.

display: none - 박스 모델을 보여지지 않도록 숨김 처리합니다.
따라서 공간을 차지하지도 않습니다.

display: inline-block - inline인데 내부는 block 처럼 표시합니다.
즉, 박스 모델이 inline 처럼 수평으로 정렬 됩니다.

h6 제목 = 폰트 사이즈(body) × 비율(1.2)
h5 제목 = 폰트 사이즈(h6) × 비율(1.2)

h4 제목 = 폰트 사이즈(h5) × 비율(1.2)

h3 제목 = 폰트 사이즈(h4) × 비율(1.2)

h2 제목 = 폰트 사이즈(h3) × 비율(1.2)

h1 제목 = 폰트 사이즈(h2) × 비율(1.2)
