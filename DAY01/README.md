##2. 그리드 시스템 설계

##3.웹 서비스를 위한 사전 협의
	- 다양한 디바이스 환경
	- 디바이스 지원 범위
	- 웹디자인 중단점 찾기
	- 웹디자인 중단점 설정
	- 다양한 웹브라우저와 운영체제 환경
	- 지원 브라우저 설정(크로스 브라우징)

##4. 그리드 시스템 디자인 제작 시 중단점 설정 방법
	1. (컬럼 폭 * 컬럼 수) + (거터 폭 * (컬럼 수 - 1) = 중단점(max-width)
	2. 중단점(max-width) + (거터 폭 * 2(margin)) = 그리드시스템 디자인 너비 폭
